# Latex-Report-Template-for-SEU
本项目维护SEU实验报告Latex模板
目前模板为作者自建通用模板，欢迎提交各课实验专用模板。
通用模板目前已定义封面、代码块、页边距等，使用clone本仓库并自行修改`report.tex`文件。封面logo在`figures`中勿删除。
